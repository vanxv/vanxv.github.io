3.支持python ruby java

vim ~/.bash_profile
source ~/.bash_profile

export ANDROID_HOME=/Users/VANXV/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/tools/:$ANDROID_HOME/platform-tools
export PATH=$PATH:$ANDROID_HOME/tools
export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.8.0_131.jdk/Contents/Home

how to Not concentrate

Veröffentlicht am 2017-07-07

#注意力

怎么样不集中注意力，注意力=脑回路，
注意力是有限的,而非取之不尽也做不完的资源
《哈佛商业评论》网站5月份的文章：《You brain only task so much focus》 史里尼皮莱（srini pillay） 哈佛大学医学院兼职助理教授

#耗能：

注意力与意志力有关
被动：意志力+注意力（做不喜欢的事）
主动：低耗能注意力（和喜欢的人聊天，看比赛，做喜欢的事）

我们在执行任务的时候，集中注意力处理时间，你会感觉有点累。这就是大脑处于工作状态

#耗能bug：

注意力消耗光后，

  1. 会出现不愿意和别人合作
  2. 不愿意帮助别人，
  3. 不能做处正确的决策。
  4. 情绪失控，如果想悲观的事情，会加速大脑失控。直接大bug(考虑用woop)

耗能值：

  1. 默认模式网络：
    1. 状态：什么事情都不做，今晚吃什么，天马行空的想问题
    2. 用处：挖掘记忆，过去与未来的思考+不同的想法链接在一起=创造力。提高重要感和理解他人。
    3. 耗能：20%的耗能
  2. 工作模式大脑
    1. 状态：执行任务
    2. 耗能25%

我们的大脑平均46.9%时间处于默认模式网络

#回血：默认神经网络(default mode network)

  1. 积极的建设性的白日梦（ positive constructive daydreaming=PCD)

    1. 做最无聊的事：织毛衣、散步、花花草草、让大脑放松下来;
    2. PCD=目标 要有目标的做白日梦 ;
    3. 想一些好玩的事儿，树林里穿梭、游艇上躺着之类的纯碎白日梦。想想你期待的事儿。;
    4. 增加创造力与领导力

  2. 睡眠：

    1. 打盹10分钟 恢复20%;
    2. 睡90分钟 恢复100%;

  3. 假装自己是专家或老师

    1. 费曼技巧
    2. 假装后自己胡思乱想一下，思维会很跳跃的解决问题。

总结：

  1. 要有攻击力，首先得学会回血，（番茄工作法和woop都有效）;
  2. 用woop把被动改成主动（李笑来说的，赋予意义与价值）（游戏化奖赏机制）;
  3. 白日梦技能加点给目标;
  4. 打坐也可以回血用敲击术;
  5. 增加脑回路可以增加注意力;
