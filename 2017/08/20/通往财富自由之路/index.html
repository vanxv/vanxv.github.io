#定义财富自由之路

1.为什么正确定义概念？

还记不记得很多人在读书的时候说，我长大了一定要做一个千万富翁的梦想？
身边是不是有很多人说，我要多么多么努力财务自由？
这个逻辑是合理的，也是有机会达到的。
对。是有机会，并不是一定。
这个坑从小学到后来一致伴随着我，走到了现在。
得到了一个公式 努力 + 梦想 = 财务自由

可为什么我们努力了那么多年，梦想还是那么遥远呢？

绝大多数努力的人（包括我自己也犯了这个大错误）

我们从来没有清晰的定义过这个概念，*什么是财务自由，他需要具备什么条件才能达到，

我们的大脑中没有一个清晰的目标，好比去的地方没有搞清楚乱跑。
因为这条路是所有男人都想走的路，而82法则无情的在财务自由的世界中流动。
我们得先弄明白。什么是财富自由，财富自由的本质是什么？

大脑中没有正确的定义，
大脑出现 混乱 -> 跑偏 -> 浮躁

##理论

个人对财富自由的定义
财富：个人(知识+信息)值 注意力(单位时间)价值 世界连接数（满足用户需求/提高效率）
自由：财富的收益率 - (通货膨胀率 +消费)= 自由
价值：满足用户需求/投资
生产：

1. 获取信息与需求（知识积累+用户连接+创造)
2. 具备技能(或雇佣)
3. 1+人 创造流程或产品
4. 推向市场

所以我们得从新定义财富自由
财富 = 个人(知识+信息)值 注意力(单位时间)价值 世界连接数（满足用户需求/提高效率）
自由 = 财富成长率>支出

知识 = 各个领域系统性信息量 * 连接创造率
例：技能（读书、写字、软件开发)+营销=利润
技能（读书、写字、软件开发)+营销+认知心理学 = 更高的利润
技能（读书、写字、软件开发)+营销+认知心理学 + 财务入门知识 = 更高更系统利润
利润大小 = 集体/个人价值 - 对手的值
你在每一个领域都有入门知识就成了一位独立的ceo

人生枷锁：影响注意力和收益率的认知

资本：有效资本周转成长率与未来价值。

落后：成长率+注意力使用率+大脑概念<99%可能已经等于落后（参考82法则／82法则中的82法则）
