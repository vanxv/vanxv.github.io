#案例

##Wish(WOOP) + Goal《游戏改变世界》

1. python全栈工程师
2. 自动化工作

##OutCome + Goal《游戏改变世界》 + 《巅峰表现》

1. 收入增加 生活更美好
2. 成为深度学习工程师
3. 可以把认知心理学和深度学习连接在一起
4. 可以帮助客户更系统的提高效益（**《巅峰表现》**）
5. 成为机械师
6. 增加财务收入速度
7. 践行可以改变身边的人，践行与知一起（**《巅峰表现》**）

##obstacle：《WOOP》 + 《游戏改变世界》

1. 看不懂英文 
2. 没有成就感
3. 不会写
4. 写得慢
5. 没有成就感
6. 没有注意力
7. 有bug
8. 太投入出不来
9. 睡不好
10. 仪式感

##Plan计划

If 看不懂英文 
    when 呼吸3秒，然后把自己会的挑出来，如果google翻译，然后把不懂的单词放入ANKI
    +  恭喜英文单词+1，并积分榜+1
    + 《游戏改变世界》记分板+1分
IF 没有成就感
    when 写WOOP，可视化WOOP 
    +  想一想你的客户，想一想你的父母，想一想你的朋友。
    +  积分榜消除焦虑记分板 +1分
IF 不会写
    When 查google，并看官方文档
    + 积分榜习得python 爬虫+1
IF 看不懂
    When 在QQ群问给大神红包 **《参考如何提问》**
    + 解决问题奖积分榜+1
IF 心情不好
    when 设置20-30分钟时间出去吃喝玩乐。
    + 解决完后。
IF 觉得自己解决不了，
    When 翻开ANKI和自己的笔记，以及自己会的东西，你还有什么不会的？
    + 自我鼓励奖 积分榜+1 
IF 心情不好设定时间
    When 吃自己想吃的，喝自己想喝的。
    + 搞定心情 +1分
IF 太投入出不来
    When 50-90分钟休息一次 
    + 最佳巅峰表现奖 + 1分
IF 开始工作
    When 仪式感
    + 仪式奖励 +1分

##Rules规则

1. 定时间
    1. 1天完成 10分
    2. 2天完成 5分
    3. 3天完成 1分
2. 定奖励
    1. 学会英语单词 1分
    2. 解决bug 2.5分
    3. 心情不好解决 3分
    4. 察觉有bug的地方 奖励 10分
    ......

3. 得分奖励表：
    得 10分 巧克力 1枚
    得 20分 中午加餐 
    得 50分 奖励1万挥霍

#最后重点中的重点 - 可视化 WOOP + 游戏规则 + 愿景

可视化WOOP和积分榜，随时抬头可视化，
