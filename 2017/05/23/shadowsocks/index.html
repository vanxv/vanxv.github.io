shadowsocks

shadowsocks是一个科学上网的工具
下载地址：
http://www.uudaili.org/windows-shadowsocks.html

http://ossfiles.fyvps.com/ssclient/Shadowsocks-win-2.5.zip
需要安装 .NET Framework 3.5 版本,推荐 win7 和 XP 客户使用
http://ossfiles.fyvps.com/ssclient/Shadowsocks-win-dotnet4.0-2.5.zip
需要安装 .NET Framework 4.0 版本,推荐 win8 和 win10 客户使用
http://ossfiles.fyvps.com/ssclient/ShadowsocksR-win-3.8.9.0e.7z
ShadowsocksR 版本支持 混淆和自定义参数

下载完成后，解压,就一个文件,很简单：

shadowsocks

双击这个文件，然后在你的任务栏右下角就会有个白色的小飞机。

shadowsocks任务栏

注意看任务栏右下角，不用一直双击，要不会打开很多个。
右击，然后选择服务器 –编辑服务器.如图：
