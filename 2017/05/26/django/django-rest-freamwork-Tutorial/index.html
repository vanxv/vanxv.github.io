Veröffentlicht am 2017-05-26

djangoRestFramwork-Url
djangoRestFramwork是一个非常强大的灵活的工具包对于绑定在网页的APIS

一些原因你将要可以使用RestFramework：

  1. 这个网页浏览是 一个巨大的可用的开发者工具
  2. 验证策略包括OAuth1a和OAuth2的包。
  3. 支持ORM和非ORM数据源的序列化。
  4. 如果不需要功能更强大的功能，可以自定义一切 - 只需使用基于功能的常规视图。
  5. 广泛的文档和极好的社区支持。

##环境要求：

  1. python（2.7 3.*)
  2. Django(1.8+)

##需要的安装包:

  1. coreapi(1.32.0)模式生成支持。
  2. markdown(2.1.0+)Markable支持可浏览的API。
  3. django-filter(1.0.1+)搜索支持
  4. django-crispy-forms 修改显示
  5. django-guardian(1.1.1+)对象级权限支持。
