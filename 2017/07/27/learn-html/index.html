##实验目的：

用anki做提取记忆模式，

anki+认知学习法+记单词关联性学习。

##逆向学习：

学习如果不能提取，我们的信息在短期记忆就被销毁，
因此要存储在长期记忆中，我们需要从践行出发制定学习学习流程
践行>提取可践行数据>数据化学习过程>实验模式测试自己的学习结果，
然后制定学习计划，按计划学习提取内容。打造自己ROI最好的学习系统。
